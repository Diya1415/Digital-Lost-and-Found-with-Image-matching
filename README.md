
# Digital Lost & Found With Image Matching

## Overview
A platform that enables users to upload photos of lost or found items and automatically matches them using image feature recognition. Notifications are sent when potential matches are discovered, making item recovery fast and simple.

## Features
- Upload and report lost/found items with images
- Automated image matching and categorization
- Real-time notifications for matches
- Searchable lost and found database

## Tech Stack
- React/Next.js for frontend
- Node.js backend on Vercel
- Bolt AI or Vercel AI for image matching
- Firebase/MongoDB for database
- Email/SMS notifications via Twilio
